Olen 23-vuotias oululainen ohjelmointiin intohimoisesti suhtautuva nuori. Sosiaalisuuteni ja kuuntelevaisuuteni auttavat minua ymmärtämään ihmisten tarpeita ja kääntämään ne toimiviksi digitaalisiksi ratkaisuiksi. Olen ahkera tekijä, joka vie projektit päätökseen asti, sillä annan aina kaikkeni tehdäkseni työstäni parhaan mahdollisen. Olen itseoppinut ohjelmoija, ja olen tehnyt useita projekteja eri koodikielillä, kuten React, React Native, JavaScript, HTML, CSS ja Java. Näistä kielistä olen erityisesti ihastunut Reactiin ja React Nativeen niiden monipuolisuuden ja tehokkuuden vuoksi. Lisätietoja ja esimerkkejä työstäni löytyy portfoliostani "Portfolio" kohdasta.
